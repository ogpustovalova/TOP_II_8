# Модуль 4. Сверточные нейронные сети

Раздел РПД №4. Сверточные нейронные сети (CNN) для анализа изображений: слои
свёртки и подвыборки, классические архитектуры (AlexNet, VGG16), проблема
исчезающего градиента и её решение в ResNet (остаточные соединения, batch
normalization), использование предобученных сетей и перенос обучения
(transfer learning).
