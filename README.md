# Фонд оценочных средств по дисциплине «Основы нейронных сетей»

Этот репозиторий — открытое и воспроизводимое представление фонда оценочных
средств (ФОС) по дисциплине «Основы нейронных сетей» (УрФУ). Здесь объединены
рабочая программа дисциплины, модель измерения результатов обучения,
контрольно-измерительные материалы (КИМ), критерии оценивания, методические
рекомендации и используемые ресурсы. Результаты обучения связываются с
компетентностно-ролевой моделью в области искусственного интеллекта (КРМ v3.0).

## 1. О дисциплине

**Цель дисциплины.** Сформировать практические навыки применения и разработки
архитектур глубоких нейронных сетей для решения прикладных задач — от
математических основ обучения до специализированных архитектур для изображений,
последовательностей и генеративных моделей.

**Основные задачи.** Освоить математический аппарат НС (нейрон, активации,
градиенты, оптимизация, backprop); научиться обучать сети (функции потерь,
SGD, регуляризация, борьба с переобучением); изучить полносвязные, сверточные,
рекуррентные сети и автокодировщики; освоить современные оптимизаторы и перенос
обучения.

**Структура и содержание.** Дисциплина состоит из 7 модулей по разделам РПД:
(1) искусственные НС, (2) обучение НС, (3) полносвязные НС, (4) сверточные НС,
(5) НС для анализа последовательностей, (6) автокодировщики, (7) алгоритмы
обучения НС. Обучение включает лекции, практические занятия и завершается
зачётом (5 семестр).

**Пререквизиты и связи с другими дисциплинами.** Теория вероятностей,
математическая статистика, машинное обучение, программирование (Python, NumPy).
Постреквизиты: компьютерное зрение, обработка естественного языка, параллельное
программирование и программирование на GPU.

**Планируемые результаты обучения.** По завершении дисциплины обучающийся сможет:
объяснять и применять математические основы НС; строить и обучать полносвязные
сети для регрессии и классификации; применять и настраивать сверточные сети для
анализа изображений, включая перенос обучения; применять рекуррентные и
одномерные свёрточные сети для текстов и временных рядов; применять
автокодировщики (AE, VAE) для снижения размерности, обнаружения аномалий и
генерации данных; сравнивать и настраивать современные оптимизаторы.

Полное описание приведено в [РПД](docs/rpd.md).

**Целевые роли КРМ** (из «Модели выпускника» программ УрФУ — см.
[`docs/urfu-program-facts.md`](docs/urfu-program-facts.md)):

- **АИИ (09.03.01 «Алгоритмы ИИ»):** ML Engineer, Data Analyst.
- **ПИИ (09.03.03 «Прикладной ИИ»):** ML Engineer, Data Engineer, MLOps.

Общая роль для обеих программ — **ML Engineer**.

## 2. Модель измерения

> Каждая строка — связка «компетенция — индикатор — уровень» (КИУ) по КРМ v3.0.
> Уровень здесь означает результат, наблюдаемо проверяемый обязательным заданием,
> а не итоговый уровень профиля выпускника. Все КИУ текущей версии ФОС заявлены
> на базовом уровне Б. Предметные дескрипторы выделяют проверяемое действие из
> КИУ; полные редакционно нормализованные формулировки КРМ приведены в
> [`data/krm-descriptors.md`](data/krm-descriptors.md). Различие между уровнями
> ФОС и целевым профилем программы пояснено в
> [`docs/urfu-program-facts.md`](docs/urfu-program-facts.md).
> Строки модулей связывают обязательное действие с одним КИУ. Лекционная
> контрольная и зачёт выборочно агрегируют ранее подтверждённые КИУ и не создают
> новые практические свидетельства; проект подтверждает только КИУ, явно
> выбранные в согласованном плане.

**Компетенции КРМ v3.0 (формулировки):**

- **DL-1.** Способен применять и (или) разрабатывать архитектуры глубоких нейронных сетей.
- **DL-3.** Способен применять и (или) разрабатывать алгоритмы, методы и технологии
  компьютерного зрения.
- **DL-4.** Способен применять и (или) разрабатывать алгоритмы, методы и технологии
  обработки естественного языка.

**Уровни КРМ:** Б — базовый, С — средний, П — продвинутый. Текущая версия ФОС
измеряет базовый уровень Б. Более высокие уровни из «Модели выпускника» УрФУ
относятся к итоговому профилю программы и не приписываются одной дисциплине без
отдельных наблюдаемых заданий.

## 4. Итоговая оценка

Правила суммирования баллов, минимальные пороги и перевод результата в принятую
шкалу оценок приведены в рубриках соответствующих КИМ и в
[методических указаниях преподавателям](methodical-guidelines/teachers-assessment/README.md).
Итог складывается из четырёх равновесных компонентов:

| Компонент | Вес |
|---|:---:|
| Среднее по 7 лабораторным | 25 % |
| Проектная работа с защитой | 25 % |
| Лекционная контрольная: `К = (К1 + К2) / 2` | 25 % |
| Зачёт | 25 % |

Формула: `И = 0,25 × Л + 0,25 × ПР + 0,25 × К + 0,25 × З`. К зачёту
допускаются обучающиеся, сдавшие все лабораторные, проект и оба рубежных среза
при `Л ≥ 40`, `ПР ≥ 40`, `К1 ≥ 40` и `К2 ≥ 40`. Для положительного результата
необходимы `З ≥ 40` и `И ≥ 40`.

Оценивание ведётся по **единой 100-балльной шкале БРС УрФУ** (см.
[`docs/urfu-program-facts.md`](docs/urfu-program-facts.md)): Высокий (В) — 80–100
(«отлично»), Средний (С) — 60–79 («хорошо»), Пороговый (П) — 40–59
(«удовлетворительно»), Недостаточный (Н) — < 40 («неудовлетворительно»). Порог
зачёта — 40 баллов. Диапазон 40–59 является положительным и не требует
пересдачи.
