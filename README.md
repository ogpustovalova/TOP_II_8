# Фонд оценочных средств по дисциплине «ОСНОВЫ ГЛУБОКОГО ОБУЧЕНИЯ»

> **Статус:** шаблон репозитория. 

Этот репозиторий предназначен для открытого и воспроизводимого представления фонда оценочных средств (ФОС) по дисциплине «ОСНОВЫ ГЛУБОКОГО ОБУЧЕНИЯ». Здесь объединены рабочая программа дисциплины, модель измерения результатов обучения, контрольно-измерительные материалы (КИМ), критерии оценивания, методические рекомендации и используемые образовательные ресурсы. Результаты обучения связываются с компетентностно-ролевой моделью в области искусственного интеллекта (КРМ версии 3.0).

## 1. О дисциплине

**Цель дисциплины.** Сформировать у студентов фундаментальные знания о теоретических основах глубокого обучения и практические навыки разработки, обучения и применения глубоких нейросетей для решения прикладных задач в области компьютерного зрения, обработки естественного языка, анализа данных и генеративных моделей. Дисциплина также направлена на развитие компетенций в выборе архитектур (свёрточные, рекуррентные, трансформеры), оптимизационных алгоритмов, методов регуляризации и оценки качества моделей, что позволит выпускникам самостоятельно проектировать и внедрять современные системы искусственного интеллекта.

## 7. Порядок заполнения

1. Заполните РПД и сформулируйте проверяемые результаты обучения.
2. Выберите релевантные профессиональные роли и компетенции КРМ.
3. Для каждого результата выберите индикатор и требуемый уровень.
4. Заполните модель измерения.
5. Создайте КИМ и рубрики оценивания по шаблонам.
6. Согласуйте систему итогового балла.
7. Добавьте ресурсы, сведения о команде и лицензию.
