### Как добавить лицензию на GitHub

1. Открыть главную страницу репозитория.
2. Выбрать **Add file → Create new file**.
3. Указать имя файла `LICENSE` или `LICENSE.md`.
4. Нажать **Choose a license template**.
5. Выбрать подходящую лицензию.
6. Нажать **Review and submit**, а затем **Commit changes**.

Если нужной лицензии нет среди шаблонов GitHub, необходимо скопировать её официальный текст в файл `LICENSE`.

В основном файле `README.md` следует указать выбранную лицензию:

```markdown
## Лицензия

Материалы репозитория распространяются на условиях лицензии
[Creative Commons Attribution 4.0 International](LICENSE).

При использовании и адаптации материалов необходимо указывать авторов
и источник.
```

Если репозиторий содержит программный код, для него рекомендуется отдельно выбрать программную лицензию, например MIT или Apache License 2.0. В этом случае можно использовать два файла:

* `LICENSE-CONTENT.md` — лицензия Creative Commons для учебных и методических материалов;
* `LICENSE-CODE.md` — лицензия для программного кода.
