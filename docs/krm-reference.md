# Краткий справочник КРМ версии 3.0

Этот файл сформирован по приложенной таблице КРМ и предназначен для навигации при проектировании дисциплины. Полные формулировки индикаторов и уровней находятся в исходной книге [`data/krm-v3.0.xlsx`](../data/krm-v3.0.xlsx).

## Уровни и обозначения

| Обозначение | Значение |
|:---:|---|
| Б | базовый уровень |
| С | средний уровень |
| П | продвинутый уровень |
| + | вспомогательная компетенция |
| * | сквозная / контекстно применимая компетенция |
| - | не является типовой для роли |

## Профессиональные роли

| Роль | Определение | Фокус компетенций | Функция в ИИ-проекте |
|---|---|---|---|
| Data Engineer (Инженер по данным) | Специалист, отвечающий за создание и поддержку инфраструктуры для сбора, обработки и хранения больших объемов данных | Технологии баз данных, инфраструктура данных, программирование (Python, Java/Scala) | Создание и поддержка инфраструктуры сбора, обработки и хранения данных |
| Data Architect (Архитектор данных) | Стратегический специалист, проектирующий общую архитектуру данных в организации и управляющий знаниями компании | Архитектура данных, управление знаниями, стратегическое планирование | Проектирование архитектуры данных в организации, обеспечивающих формирование их ценности методами ИИ |
| AI Architect (Архитектор ИИ) | Специалист по проектированию технической архитектуры систем ИИ, выбору технологического стека и интеграции ИИ-компонентов | Системное проектирование, архитектура ИИ, техническое лидерство | Проектирование архитектуры систем ИИ, определение подходов и технологий для реализации проекта |
| ML Researcher (Исследователь МО) | Ученый-исследователь, разрабатывающий новые алгоритмы машинного обучения и проводящий фундаментальные исследования в области ИИ | Фундаментальная математика, исследовательские методы, инновационные алгоритмы | Генерация новых фундаментальных знаний, создание новых алгоритмов машинного обучения и моделей ИИ |
| ML Engineer (Инженер МО) | Инженер, специализирующийся на практической реализации и промышленном внедрении моделей машинного обучения | Практическое МО, промышленная разработка, оптимизация систем | Практическая реализация и промышленное внедрение моделей машинного обучения в системах ИИ |
| AI PM (Менеджер проектов ИИ) | Продуктовый менеджер, управляющий жизненным циклом ИИ-продуктов и координирующий междисциплинарные команды | Управление проектами, бизнес-анализ, координация команд | Управление процессами создания ИИ-решений, включая координацию команды разработки |
| Data Analyst (Аналитик данных) | Специалист по анализу данных, извлечению инсайтов и построению аналитических моделей для поддержки бизнес-решений | Статистический анализ, визуализация данных, предварительная обработка | Извлечение знаний из данных, построение аналитических моделей, использующих МО и ИИ |
| AI CDO (Руководитель цифровой трансформации) | Стратегический руководитель, отвечающий за цифровую трансформацию организации с использованием технологий ИИ | Стратегическое планирование, цифровая трансформация, лидерство | Проектирование новых путей формирования ценности в организации (отрасли) посредством применения ИИ |
| AI Security Engineer (Инженер безопасности ИИ) | Специалист по обеспечению безопасности, надежности и этичности систем искусственного интеллекта | Безопасность ИИ, этика ИИ, управление рисками | Обеспечение доверенности (безопасности, надежности, этичности и пр.) систем ИИ |
| TADS (Технический аналитик данных) | Технический специалист, сочетающий навыки анализа данных с пониманием технических аспектов их обработки и хранения | Техническая аналитика, обработка данных, поддержка ML-проектов | Техническая реализация процедур хранения, обработки и анализа данных |
| MLOps (Специалист по эксплуатации ИИ) | DevOps-инженер, специализирующийся на автоматизации и операционном управлении жизненным циклом ML-моделей | DevOps для ML, автоматизация, мониторинг систем | Автоматизация и операционное управление жизненным циклом MО-моделей |
