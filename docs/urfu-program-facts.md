## 4. Единая шкала БРС УрФУ (из всех 9 изученных ФОС)

100-балльная шкала, единая для всех дисциплин УНЦ «ИИ»:

| Уровень | Баллы | Традиционная оценка | Зачёт | Качественный уровень |
|---|---|---|---|---|
| 1 | Достигнуты в полном объёме | **Отлично (80–100)** | Зачтено | **Высокий (В)** |
| 2 | В целом достигнуты, есть замечания | **Хорошо (60–79)** | Зачтено | **Средний (С)** |
| 3 | Достигнуты не в полной мере | **Удовлетворительно (40–59)** | Зачтено | **Пороговый (П)** |
| 4 | Существенные ошибки | **Неудовлетворительно (< 40)** | Не зачтено | **Недостаточный (Н)** |

**Порог зачёта = 40 баллов.** Уровни шкалы — **В/С/П/Н** (не Б/С/П).

## 5. Структура ФОС УрФУ (из образцов)

Документ называется «ОЦЕНОЧНЫЕ МАТЕРИАЛЫ ПО ДИСЦИПЛИНЕ…», 5 разделов:

1. **Структура и объём дисциплины** (таблица: модуль, з. е., виды занятий, формы аттестации).
2. **Планируемые результаты обучения (индикаторы)** (компетенция / индикатор / контрольно-оценочные средства).
3. **Процедуры контроля в БРС**:
   - 3.1. Технологическая карта БРС по семестрам (коэффициенты значимости по видам занятий, в сумме 1.0; внутри каждого — текущая 0.5 / промежуточная 0.5).
   - 3.2. Аттестация курсовой работы/проекта (обычно «не предусмотрено»).
4. **Критерии и уровни оценивания** (Знания/Умения/Опыт/Другие результаты + универсальная шкала В/С/П/Н).
5. **Содержание контрольно-оценочных мероприятий** (КИМ):
   - 5.1. Аудиторные (лекции, лабораторные/практические).
   - 5.2. Внеаудиторные (контрольные, домашние работы по вариантам).
   - 5.3. Промежуточный контроль (зачёт и/или экзамен со списком вопросов).

## 6. Типовая технологическая карта БРС (для ОНС)

По образцу ФОС «Машинное обучение», «ЦИНИ» (лекции + лабораторные):

- Лекции — коэффициент 0.5 (текущая аттестация 0.5 + промежуточная 0.5).
- Лабораторные — коэффициент 0.5 (текущая 1.0 внутри; каждое задание ≤ 100 баллов).
- Курсовая работа/проект — «не предусмотрено» (если нет отдельного курса).

Точные коэффициенты для ОНС требуют решения автора курса. Текущая версия ФОС
предлагает встроить проект в текущую аттестацию лабораторного блока: среднее
лабораторных, проект, лекционная контрольная и зачёт получают по 25 %.
Лекционная контрольная состоит из срезов К1 и К2, каждый из которых имеет
эффективный вес 12,5 %. Это авторская схема, а не извлечённый из исходной РПД
норматив.
