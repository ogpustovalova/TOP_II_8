## Релевантные листы для курса «Основы нейронных сетей»

Курс опирается на компетенции DL-1, DL-3, DL-4 из листа «Профессиональные» и
матрицу уровней из листа «Роли-компетенции наглядно». Целевые роли курса (из
«Модели выпускника» УрФУ — см. [`docs/urfu-program-facts.md`](../docs/urfu-program-facts.md)):

- **АИИ (09.03.01):** ML Engineer, Data Analyst.
- **ПИИ (09.03.03):** ML Engineer, Data Engineer, MLOps.

## Правило использования

Не редактируйте исходную книгу при заполнении ФОС. Выбранные коды, индикаторы и
уровни сверяйте с исходной книгой. В документы дисциплины добавляйте отдельный
предметный дескриптор наблюдаемого действия, не выдавая его за полную цитату КРМ.

## Замечание о нумерации индикаторов

В РПД УрФУ использована собственная нумерация индикаторов DL-1.1–DL-1.6, которая
не совпадает с КРМ. В документах этого репозитория коды приведены в соответствие
с КРМ v3.0 (см. раздел 1 [плана заполнения](../docs/fill-plan.md)).
