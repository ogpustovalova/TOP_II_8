# Модуль 7. Алгоритмы обучения нейронных сетей

Раздел РПД №7. Алгоритмы обучения нейронных сетей: современные оптимизаторы
(SGD с моментом, Adam, AdamW, RMSProp), параметр скорости обучения и управление
им (lr scheduling), ранняя остановка. Связь выбора оптимизатора со скоростью и
качеством сходимости.
