# Модуль 6. Автокодировщики

Раздел РПД №6. Автокодировщики (автоэнкодеры, AE) и вариационные автокодировщики
(VAE): задача восстановления объекта на входе сети, скрытое (латентное)
пространство и плотные векторные представления (эмбеддинги), снижение
размерности, обнаружение аномалий, генерация данных.
