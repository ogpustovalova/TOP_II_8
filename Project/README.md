# Проектная работа

Проектная работа — интегральный контрольно-измерительный материал (КИМ) по
дисциплине «Основы нейронных сетей». Она проверяет способность обучающегося
связать знания из нескольких модулей в единую прикладную или исследовательскую
задачу: от постановки и выбора архитектуры до обучения, валидации, анализа
ошибок и защиты результата. Проект оценивается по 100-балльной рубрике и даёт
25 % итоговой оценки по дисциплине.

## КИУ (КРМ v3.0)

| Компетенция | Индикатор | Уровень |
|---|---|:---:|
| DL-1. Архитектуры глубоких НС | DL-1.1 — математические основы и backprop | Б |
| DL-1 | DL-1.2 — многослойный персептрон | Б |
| DL-1 | DL-1.3 — современные архитектуры глубоких сетей | Б |
| DL-1 | DL-1.4 — спец. архитектуры для изображений | Б |
| DL-3. Компьютерное зрение | DL-3.1 — алгоритмы и библиотеки CV | Б |
| DL-4. Обработка естественного языка | DL-4.1 — алгоритмы и библиотеки NLP | Б |

Вариант 1 может подтверждать DL-1.3_Б, DL-1.4_Б, DL-3.1_Б и DL-4.1_Б в
зависимости от предметной области. Вариант 2 может подтверждать DL-1.1_Б,
DL-1.2_Б и DL-1.4_Б. Конкретный проект подтверждает только КИУ, явно
зафиксированные в согласованном плане и обеспеченные обязательным артефактом;
выбор варианта сам по себе не подтверждает весь допустимый набор.
