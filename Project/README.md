# Проектная работа

Проектная работа — интегральный контрольно-измерительный материал (КИМ) по
дисциплине «Основы нейронных сетей». Она проверяет способность обучающегося
связать знания из нескольких модулей в единую прикладную или исследовательскую
задачу: от постановки и выбора архитектуры до обучения, валидации, анализа
ошибок и защиты результата.

## КИУ (КРМ v3.0)

| Компетенция | Индикатор | Уровень |
|---|---|:---:|
| DL-1. Архитектуры глубоких НС | DL-1.3 — современные архитектуры глубоких сетей | Б (→С) |
| DL-1 | DL-1.4 — спец. архитектуры для изображений | Б |
| DL-3. Компьютерное зрение | DL-3.1 — алгоритмы и библиотеки CV | Б |
| DL-4. Обработка естественного языка | DL-4.1 — алгоритмы и библиотеки NLP | Б |
