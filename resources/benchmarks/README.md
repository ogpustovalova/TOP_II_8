## Зачем в курсе

Бенчмарки используются как: (1) источник предобученных весов для transfer
learning (ImageNet → ResNet); (2) стандартизованная площадка для сравнения
архитектур и оптимизаторов (CIFAR-10). Глубокого участия в бенчмарках курс не
предполагает — это вводный уровень.
