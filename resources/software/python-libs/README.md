## Требования к добавлению

Для каждой библиотеки укажите:

- **Версию** – фиксированную или минимальную версию, с которой гарантирована работа примеров.
- **Команду установки** – например, `pip install numpy==1.24`.
- **Назначение** – краткое описание, для чего используется библиотека в курсе.
- **Совместимость** – с другими библиотеками, версия Python (например, Python ≥ 3.8).
- **Лицензию** – тип лицензии, позволяющий использовать в образовательных целях.
- **Пример минимального использования** – простой фрагмент кода, демонстрирующий основные возможности.

**Пример описания для PyTorch:**

- **Версия:** ≥ 2.0.0.
- **Команда установки:** `pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118` (для CUDA 11.8) или `pip install torch torchvision torchaudio` (для CPU версии).
- **Назначение:** Основной фреймворк для глубокого обучения. Используется во всех практических заданиях.
- **Совместимость:** Python ≥ 3.8, поддержка CUDA 11.8+ / ROCm.
- **Лицензия:** BSD-3-Clause.

  import torch
