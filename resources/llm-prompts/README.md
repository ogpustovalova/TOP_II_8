# LLM-сервисы и промпты

Промпты для больших языковых моделей как учебный инструмент: объяснение
концепций, генерация учебных данных, разбор ошибок в коде. Применение LLM
сопровождается оговорками: модель может давать правдоподобные, но неверные
ответы — результаты нужно сверять с источниками.

## Правила использования

- LLM — для пояснений и отладки, **не для генерации решения КИМ целиком**.
- Любое значимое использование отмечайте в комментариях со ссылкой на промпт.
- Ответы LLM сверяйте с первоисточниками (статьи, учебник d2l.ai).
- На защите студент должен уметь объяснить решение без помощи LLM.
