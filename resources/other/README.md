## Требования к добавлению

Размещайте здесь только материалы, которые нельзя содержательно отнести к другим разделам:

- **Платформы для выполнения кода** (Google Colab, Kaggle).
- **Инструменты для управления проектами и заметками** (GitHub, Overleaf, Notion, Obsidian).
- **Агрегаторы научных публикаций** (arXiv, Papers with Code).
- **Онлайн-курсы и образовательные платформы** (Coursera, Fast.ai, Stepik).
- **Сообщества и форумы** (Stack Overflow, Telegram-каналы).
- **Инструменты для визуализации и логирования** (TensorBoard, Weights & Biases).

### Пример описания для GitHub:

- **Название:** GitHub.
- **Аннотация:** Платформа для хранения, версионирования и совместной работы над кодом. Используется для размещения лабораторных работ, проектов и обмена материалами между студентами и преподавателями.
- **Связанные КИМ:** Все модули (проекты, лабораторные работы).
- **Доступ:** [github.com](https://github.com/).
- **Лицензия / условия:** Бесплатный для публичных репозиториев; платные тарифы для приватных (доступны через GitHub Education для студентов и преподавателей).
- **Дата проверки:** 2026-07-19.

### Пример описания для TensorBoard:

- **Название:** TensorBoard.
- **Аннотация:** Инструмент для визуализации процесса обучения моделей TensorFlow (и PyTorch через `torch.utils.tensorboard`). Используется для отслеживания кривых обучения, гистограмм весов, карт внимания и других метрик в реальном времени.
- **Связанные КИМ:** Лабораторные работы (все модули).
- **Доступ:** Встроен в TensorFlow / PyTorch (`pip install tensorboard`).
- **Лицензия / условия:** Бесплатный (Apache 2.0 для TensorFlow).
- **Дата проверки:** 2026-07-19.
