# Банки тестовых заданий

Тестовые задания для рубежного контроля и самопроверки — на знание терминологии,
формул и базовых фактов. Тесты дополняют (но не заменяют) устную защиту: в эпоху
генеративного ИИ проверка терминологии тестом имеет смысл, а проверка
понимания — только через обсуждение.

## Тематические блоки

| Блок | Что проверяется | Связанные КИМ |
|---|---|---|
| Модель нейрона | формула `y=f(w·x+b)`, роли весов/смещения, виды активаций | [М1](../../M1-neuron-basics/kim-01-neuron-perceptron.md) |
| Backprop | правило цепи, производные softmax+cross-entropy, роль `zero_grad`/`backward`/`step` | [М2](../../M2-training/kim-01-backprop-training.md) |
| Регуляризация | Dropout, L2, EarlyStopping — механизмы и отличия | [М2](../../M2-training/kim-01-backprop-training.md) |
| UAT | формулировка и ограничение «существование ≠ находимость» | [М3](../../M3-dense-networks/kim-01-mlp.md) |
| Свёртка | формула размера выхода, stride, padding, pooling | [М4](../../M4-cnn/kim-01-cnn.md) |
| ResNet | residual connection, проблема затухающего градиента, BatchNorm | [М4](../../M4-cnn/kim-01-cnn.md) |
| Transfer learning | feature extraction vs fine-tuning, роль lr при fine-tuning | [М4](../../M4-cnn/kim-01-cnn.md) |
| RNN/LSTM/GRU | затухание градиента, гейты, `Embedding` | [М5](../../M5-sequences/kim-01-sequences.md) |
| AE/VAE | латентное пространство, репараметризация, KL-дивергенция | [М6](../../M6-autoencoders/kim-01-autoencoders.md) |
| Оптимизаторы | SGD/Adam/AdamW/RMSProp — формулы обновления и отличия; lr scheduling | [М7](../../M7-optimizers/kim-01-optimizers.md) |

## Формат заданий

- Закрытые (с выбором ответа) — для терминов и формул.
- Открытые (короткий ответ) — для формул и определений.
- Смешанные — небольшие расчётные (например, «число параметров слоя
  `Dense(800)` при входе 784»).

## Требования к добавлению

Для каждого банка укажите темы, количество заданий, уровни сложности, правила
выборки, ключи и порядок обновления. Наполняется преподавателем по мере
развития курса.
