## Требования к добавлению

Для каждого банка укажите темы, количество заданий, уровни сложности, правила выборки, ключи и порядок обновления.

**Пример описания для внутреннего банка рубежного контроля:**

- **Темы:** Основы нейросетей (MLP, функции активации, оптимизаторы, регуляризация); Свёрточные сети (свёртка, пулинг, архитектуры, трансферное обучение); Рекуррентные и трансформерные сети (RNN, LSTM, механизм внимания, BERT/GPT); Генеративные модели и RL (GAN, VAE, Q-learning, Policy Gradient).
- **Количество заданий:** Не менее 50 вопросов (по 10–15 на модуль).
- **Уровни сложности:** Базовый (знание определений и основных принципов) — 40%; Средний (понимание и применение) — 40%; Продвинутый (анализ и синтез) — 20%.
- **Правила выборки:** Для каждого рубежного теста из банка случайным образом выбирается 15–20 вопросов, пропорционально распределённых по темам и уровням сложности.
- **Ключи:** Эталоны ответов хранятся в закрытом разделе для преподавателей. Для открытых вопросов разработаны критерии оценки.
- **Порядок обновления:** Банк обновляется ежегодно перед началом семестра: добавляются новые вопросы, актуализируются формулировки, корректируются ключи на основе анализа результатов предыдущих сессий.
