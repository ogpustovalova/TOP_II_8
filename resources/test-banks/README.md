# Банки тестовых заданий

Тестовые задания для рубежного контроля и самопроверки — на знание терминологии,
формул и базовых фактов. Письменные срезы проверяют явное обоснование решений, а
устная защита лабораторных и проекта отдельно проверяет понимание собственного
кода и самостоятельность.

## Формат заданий

- Закрытые (с выбором ответа) — для терминов и формул.
- Открытые (короткий ответ) — для формул и определений.
- Смешанные — небольшие расчётные (например, «число параметров слоя
  `Dense(800)` при входе 784»).

## Требования к расширению

Для нового варианта укажите темы, количество заданий, уровень сложности,
числовые параметры и ключ. Вариант должен сохранять структуру 20/40/(30+10) и не
повторять опубликованные формулировки дословно. Перед использованием другой
преподаватель проверяет ключ и сопоставимость сложности с готовыми вариантами.
