# Лекционная контрольная

Лекционная контрольная — рубежный КИМ по теоретическому и расчётному материалу
курса «Основы нейронных сетей». Она состоит из двух обязательных срезов:

| Срез | Срок | Темы | КИУ, понимание которых проверяется |
|---|---|---|---|
| **К1** | После модуля 3 | Нейрон, функции активации, backprop, функции потерь, регуляризация, MLP | DL-1.1_Б, DL-1.2_Б |
| **К2** | После модуля 7 | CNN/ResNet, transfer learning, последовательности, NLP, AE/VAE, оптимизаторы | Выборка DL-1.1_Б, DL-1.3_Б, DL-1.4_Б, DL-1.5_Б, DL-1.7_Б, DL-1.12_Б, DL-3.1_Б, DL-4.1_Б |

Каждый срез оценивается от 0 до 100. Компонент лекционной контрольной в БРС:

`К = (К1 + К2) / 2`.

К1 и К2 проверяют объяснение, расчёт и анализ ситуации. Они дополняют, но не
заменяют практические свидетельства из лабораторных и проекта.

При результате ниже 40 пересдаётся только соответствующий срез по новому
варианту той же спецификации.
