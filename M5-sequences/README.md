# Модуль 5. Нейронные сети для анализа последовательностей

Раздел РПД №5. Нейронные сети для данных переменной длины — текстов и временных
рядов: одномерные свёрточные сети (1D-CNN), рекуррентные нейронные сети (RNN),
их улучшенные варианты LSTM и GRU, развёртывание рекуррентной сети во времени.
Применение к задачам классификации текстов и прогнозирования временных рядов.
