# Зачёт / экзамен

Промежуточная аттестация подтверждает освоение дисциплины в целом по всем семи
модулям и не дублирует отдельный модуль. Фонд оценочных средств по дисциплине
включает текущий контроль (лабораторные работы по модулям), промежуточный
(рубежные срезы и проектная работа) и итоговый — зачёт/экзамен.

## Формат

Защита проектной работы + теоретические вопросы по всем семи модулям. Устная
форма с обсуждением выбранной архитектуры и методов обучения — это основной
надёжный контроль в эпоху широкого доступа к генеративному ИИ.
