# Зачёт

Промежуточная аттестация агрегирует результаты по всем семи модулям и не
создаёт новые КИУ вместо практических заданий. Фонд оценочных средств по дисциплине
включает текущий контроль (лабораторные работы по модулям), рубежный контроль
([срезы К1/К2](../Lecture-control/README.md) и проектную работу) и промежуточную
аттестацию — зачёт.

## КИУ (КРМ v3.0)

| Компетенция | Индикатор | Уровень |
|---|---|:---:|
| DL-1. Архитектуры глубоких НС | DL-1.1; DL-1.2; DL-1.3; DL-1.4; DL-1.5; DL-1.7; DL-1.12 | Б |
| DL-3. Компьютерное зрение | DL-3.1 | Б |
| DL-4. Обработка естественного языка | DL-4.1 | Б |

Указанные КИУ должны быть предварительно подтверждены лабораторными или
согласованным вариантом проекта. Устная часть зачета проверяет целостность
понимания и самостоятельность выполнения.

## Формат

Устный зачёт: два теоретических вопроса из разных модулей, ситуационная задача и
интегральное объяснение связей между темами курса. Защита проекта оценивается
отдельно в проектной рубрике и повторно в балл зачёта не включается.
