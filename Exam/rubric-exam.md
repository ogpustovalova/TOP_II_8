# Рубрика «Зачёт / экзамен»

Шкала: **Б** / **С** / **П**. Итоговый уровень = минимум из «Теория» и
«Защита проекта». Перевод в баллы: **Б → 60–74**, **С → 75–89**, **П → 90–100**.
Порог положительной оценки — не ниже Б.

| Критерий | Б (60–74) | С (75–89) | П (90–100) |
|---|---|---|---|
| **Теория (2 вопроса)** | Отвечает верно по сути, с небольшими неточностями | То же + точные формулировки, формулы, сравнения | То же + глубокое понимание, связи между модулями, критическая оценка методов |
| **Ситуационная задача** | Называет вероятную причину и базовое решение | То же + перечисляет несколько гипотез, предлагает план диагностики | То же + ранжирует гипотезы по вероятности, предлагает количественные проверки, обсуждает ограничения |
| **Защита проекта** | Объясняет выбор архитектуры и метрики на базовом уровне | То же + обосновывает решения, реагирует на уточняющие вопросы | То же + глубоко владеет материалом, отвечает на вопросы «почему», корректно обсуждает ограничения и альтернативы |

## Пороги ассесмента по методологии КРМ

| Статус | Баллы | Доля выполненных заданий | Решение |
|---|---|---|---|
| Достигнут | ≥ 60 | ≥ 70 % | положительная оценка по дисциплине |
| Частично достигнут | 40–59 | ≥ 50 % | пересдача в установленные сроки |
| Не достигнут | < 40 | < 50 % | повторное изучение / пересдача |

Для зачёта по дисциплине студент должен достичь уровня **не ниже Б** по каждому
из критериев.

## Уровни по КРМ

| Уровень | Баллы | Соответствие |
|---|---|---|
| П (продвинутый) | 90–100 | исчерпывающие ответы, глубокое понимание |
| С (средний) | 75–89 | точные ответы с обоснованием |
| Б (базовый) | 60–74 | верные ответы по сути, минимальный порог курса |
| Ниже Б | < 60 | не достигнут — пересдача |
