# Модуль 3. Полносвязные нейронные сети

Раздел РПД №3. Архитектура полносвязных (dense) сетей, построение и обучение для
двух основных задач машинного обучения — регрессии и классификации. Связь с
универсальной теоремой аппроксимации (UAT) и практический выбор глубины/ширины сети.
