## Вклад участников

Рекомендуется кратко описать вклад по принципу CRediT или в свободной форме:
концепция, методология, разработка КИМ, программная реализация, проверка,
редактирование, управление проектом.

Участники добавляют материалы через собственные GitHub-аккаунты: история
коммитов отражает их вклад.
