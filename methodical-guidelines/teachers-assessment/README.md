# Методические указания преподавателям по оцениванию

## Принципы

1. Оценивайте наблюдаемые результаты, а не субъективное впечатление от работы.
2. Используйте опубликованную рубрику и одинаковые критерии для сопоставимых работ.
3. Отделяйте критические ошибки от недочетов оформления.
4. Связывайте обратную связь с конкретным критерием и способом улучшения.
5. Проверяйте соответствие сложности задания заявленному уровню Б/С/П.

## Калибровка оценивания

- проведите совместную проверку 3–5 работ несколькими экспертами;
- обсудите расхождения и уточните дескрипторы рубрики;
- сохраните обезличенные примеры работ разных уровней;
- при существенном изменении КИМ повторите калибровку.

## Использование автоматической проверки

Автотесты, статический анализ и LLM-оценивание могут использоваться как вспомогательные инструменты. Итоговое решение преподавателя должно учитывать валидность тестов, возможные ложные срабатывания и предметный контекст.

## Обратная связь

Рекомендуемый формат: «критерий → наблюдение → влияние на результат → конкретный способ исправления».
