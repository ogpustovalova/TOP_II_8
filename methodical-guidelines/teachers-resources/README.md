## 1. Назначение документа

Настоящие методические указания содержат рекомендации для преподавателей по подбору и использованию учебно-методических, программных и информационных ресурсов при проведении занятий по дисциплине «Основы глубокого обучения». Материал структурирован по типам ресурсов и содержит практические рекомендации по их применению на различных этапах обучения.

## 8. Заключительные рекомендации

- **Адаптируйте выбор ресурсов под уровень группы**: для начинающих предпочтительнее Nielsen и Траск, для продвинутых — Bishop и Goodfellow.
- **Используйте русскоязычные ресурсы** (DeepMachineLearning.ru, Николенко) для студентов, испытывающих трудности с английским.
- **Поощряйте использование официальной документации** PyTorch и Hugging Face — это развивает навыки самостоятельной работы.
- **Интегрируйте Kaggle-соревнования** в учебный процесс: они повышают мотивацию и дают опыт работы с реальными данными.
- **Обновляйте список ресурсов** ежегодно, так как область глубокого обучения активно развивается.
