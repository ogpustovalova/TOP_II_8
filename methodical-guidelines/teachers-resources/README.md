# Методические указания преподавателям по использованию ресурсов

## Выбор ресурсов

Ресурс включается в ФОС, если он:

- непосредственно поддерживает выполнение или проверку КИМ;
- доступен целевой аудитории;
- имеет понятные условия лицензирования;
- достаточно устойчив и актуален;
- описан в соответствующем README таблицей аннотаций.

## Внешние материалы

Если файл нельзя размещать в репозитории, добавьте внешнюю ссылку, библиографическое описание, дату проверки доступности и пояснение, для какого КИМ он нужен.

## Датасеты и бенчмарки

Укажите источник, лицензию, структуру данных, ограничения использования, риски утечки целевой переменной, правила разбиения и метрики.

## Программные библиотеки и сервисы

Фиксируйте версию, способ установки, минимальную рабочую среду, ограничения бесплатного тарифа и альтернативы.

## LLM и промпты

Для промптов указывайте назначение, входы, ожидаемый формат выхода, ограничения, тестовые примеры и критерии проверки результата. Не используйте непроверенную генерацию как единственный источник итоговой оценки.
