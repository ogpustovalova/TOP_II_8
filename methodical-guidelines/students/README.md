# Методические указания обучающимся

## До выполнения КИМ

1. Прочитайте паспорт КИМ и проверяемые результаты обучения.
2. Изучите критерии рубрики до начала работы.
3. Убедитесь, что доступны исходные данные и требуемое программное обеспечение.
4. Уточните правила использования LLM, библиотек, чужого кода и справочных материалов.

## Во время выполнения

- сохраняйте промежуточные версии;
- фиксируйте решения, предположения и ограничения;
- проверяйте воспроизводимость результата;
- отмечайте использованные источники и внешнюю помощь;
- при командной работе фиксируйте индивидуальный вклад.

## Перед сдачей

- [ ] выполнены все пункты задания;
- [ ] формат файлов соответствует требованиям;
- [ ] код запускается в указанной среде;
- [ ] выводы согласованы с полученными результатами;
- [ ] источники и применение LLM раскрыты;
- [ ] работа проверена по рубрике.
