# Модуль 1. Искусственные нейронные сети

Раздел РПД №1. Введение в математическую модель искусственного нейрона, функции
активации и простейшие нейронные сети. Базовые понятия, на которых строится весь
курс, и первые шаги в фреймворках PyTorch и TensorFlow/Keras.
